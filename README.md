# React Starter App

![React Starter App](/src/images/React-Starter-App.gif)

## Install

Some basic Git commands are:

```
$ git clone https://github.com/mohamedsamara/React-Starter-App.git
$ cd project
$ npm install

```

## Start

npm start

## Simple build for production

npm run build

## Run test with Jest

npm run test
